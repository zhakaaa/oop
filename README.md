# Object-Oriented Programming (OOP)

This repository contains my coursework for the **Object-Oriented Programming (OOP)** course, including laboratory works and a full-scale group project completed during my studies.

## Contents

The repository includes:

* Laboratory works demonstrating key OOP concepts such as inheritance, exception handling, serialization, and comparators.
* A group project — **University Management System** implemented in Java.
* UML diagrams (use case and class diagrams) used for system design.
* Source code organized into packages for better structure and maintainability.
* Project documentation and supporting materials.

## Project Overview

The main project is a **console-based University Management System** developed in Java. It is designed to simulate and manage core university operations through different user roles, including **students, teachers, and administrators**.

Each role has specific functionalities, for example:

* Students can view grades and course-related information.
* Teachers can manage courses and evaluate students.
* Administrators have full control over users, courses, and system data.

The system was designed using OOP principles and UML diagrams, followed by implementation in Java with a strong focus on clean architecture, modularity, and data organization.

## Purpose

This repository serves as a structured collection of my practical work in OOP, as well as a demonstration of applying theoretical concepts to a real-world software project.
